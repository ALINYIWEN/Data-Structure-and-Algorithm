# Data-Structure-and-Algorithm
計算機程序是執行特定任務的指令的集合。 為此，計算機程序可能需要存儲數據，檢索數據並對該數據執行計算。
數據結構是可用於存儲和組織數據的命名位置。 並且，算法是解決特定問題的步驟的集合。 學習數據結構和算法使我們能夠編寫高效且優化的計算機程序。
